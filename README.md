# Reflection Journal - 个人习题复习工具

一个基于SM-2算法的个人习题复习管理系统，支持Markdown和LaTeX公式渲染，使用IndexedDB实现大容量本地存储。

## ✨ 核心特性

### 📚 四大功能模块

- **Review（复习）** - 基于SM-2算法的智能间隔重复复习
  - 每日定额22道题，轻量化任务
  - 优先级排序：逾期久、不熟题目优先
  - 支持加练功能，灵活可选
  - 专注模式：正面只显示题干，避免干扰思考

- **Capture（录入）** - 快速录入习题
  - 支持Markdown和LaTeX公式
  - 支持原解照片上传
  - 自定义录入原因
  - 最近录入快速查看

- **Insights（分析）** - 学习数据可视化
  - 科目分布饼图
  - 录入原因分析
  - 30天掌握趋势
  - 薄弱知识点提醒
  - 存储空间监控

- **Browse（浏览）** - 搜索、筛选、管理
  - 全文搜索
  - 多维度筛选
  - 题目编辑功能
  - 批量管理

### 🎯 智能复习系统

#### 严格归档判定
题目归档需同时满足：
- 复习间隔 ≥ 90天
- 连续成功轮次 ≥ 4次
- 最近两次反馈均为 Good/Easy

#### 优先级排序
1. 逾期时间最久的题目（优先清积压）
2. 连续答对次数最少的题目（优先攻克薄弱）
3. 同优先级随机打乱（避免背顺序）

#### 每日定额
- 基础定额：22道/天
- 加练功能：完成定额后可选加练1题或5题
- 剩余题目次日优先显示

### 💾 存储系统

- **IndexedDB存储** - 容量无限制（受磁盘空间限制）
- **自动迁移** - localStorage数据自动迁移到IndexedDB
- **数据导入导出** - JSON格式，方便备份和迁移
- **存储监控** - 实时显示存储使用情况

## 🔧 部署到GitHub Pages

1. Fork 本仓库
2. 进入仓库 **Settings** → **Pages**
3. Source 选择 **Deploy from a branch**
4. Branch 选择 **main**，Folder 选择 **/ (root)**
5. 点击 **Save**
6. 等待1-3分钟后访问 `https://你的用户名.github.io/Reflection-Journal/`

## 📝 技术栈

- **前端**：纯HTML + JavaScript，无需构建
- **样式**：Tailwind CSS v3 (CDN)
- **Markdown**：Marked.js
- **LaTeX**：KaTeX
- **图表**：Chart.js
- **算法**：SM-2 间隔重复算法
- **存储**：IndexedDB（支持降级到localStorage）

## 📖 使用说明

### 录入题目
1. 切换到 **Capture** 标签
2. 填写题目信息：
   - 科目（如：高等数学）
   - 来源（如：教材P123）
   - 知识点（多个用逗号分隔）
   - 题干（支持Markdown和LaTeX）
   - 正确答案
   - 录入原因（思路错误/解法不会/难题/好题）
   - 原解照片（可选）
3. 点击保存

### 复习题目
1. 打开 **Review** 标签，自动显示今日待复习题目
2. 查看题干，专注思考解答
3. 点击"显示答案"查看：
   - 详细信息（科目、来源、知识点）
   - 正确答案
   - 原解照片（如有）
4. 根据回忆情况选择反馈：
   - **Again** - 完全忘记，1天后复习
   - **Hard** - 困难，间隔×1.2
   - **Good** - 良好，间隔×2.5
   - **Easy** - 简单，间隔×4.0
5. 完成22道基础定额后，可选择加练或结束

### 查看统计
- **Insights** 标签显示：
  - 总题目数、活跃复习中、总复习次数、平均正确率
  - 存储使用情况
  - 科目分布饼图
  - 录入原因分析
  - 30天掌握趋势折线图
  - 薄弱知识点提醒（Again率>40%）

### 管理题目
- **Browse** 标签支持：
  - 全文搜索题干或来源
  - 按科目、录入原因、知识点筛选
  - 点击题目查看详情
  - 编辑题目信息
  - 删除题目

### 数据管理
- **导出数据**：下载JSON备份文件
- **导入数据**：从JSON文件恢复数据
- **清空数据**：删除所有题目（需二次确认）

## 🎨 特色功能

### Markdown + LaTeX 支持
```markdown
设 $f(x)$ 在 $\mathbb{R}$ 上连续，求极限：

$$\lim_{x \to 0} \frac{\sin x}{x} = 1$$
```

### 智能知识点分析
- 自动统计每个知识点的复习次数
- 计算Again率（遗忘率）
- 识别薄弱知识点（Again率>40%）
- 提供针对性复习建议

### 深色模式
- 自动跟随系统主题
- 手动切换深色/浅色模式
- 图表颜色自适应

## 📊 存储说明

### IndexedDB 优势
- **容量无限制**：不再受localStorage 5MB限制
- **支持大文件**：可存储大量图片
- **性能更好**：异步操作，不阻塞UI
- **持久化存储**：关闭浏览器后数据仍在

### 数据安全
- 数据存储在本地浏览器中
- 不会上传到服务器
- 按域名隔离，其他网站无法访问
- 清除浏览器数据会删除（建议定期备份）

## 🔄 数据迁移

如果你之前使用localStorage版本：
1. 打开新版本页面
2. 自动检测并迁移旧数据
3. 迁移成功后显示提示
4. 旧数据自动清理
